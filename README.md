## Project Overview
901FitHub is a fitness digital product store concept designed to sell curated meal plans and workout programs. The application supports browsing digital products, managing content, and organizing fitness resources in a user-friendly web interface.

- Project Type: Team-based academic web application  
- Duration: One semester  
- Team Size: 3

## How to Try the App
This application is deployed as a live demo for portfolio and demonstration purposes.

You can:
- Browse workout and meal plans
- Create an account using a fake email address
- Complete a checkout flow using dummy credit card information
- Download real PDF workout and meal plans after checkout

**No real payments are processed.**

## Demo Checkout Notes

## How to Run the App Locally

```bash
bundle install
rails db:create db:migrate
rails s
```

Then visit `http://localhost:3000` in your browser.
